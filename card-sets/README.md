# Card Sets

Example card images for the picture matching game workshop. Participants upload a set to a Google Drive folder, then connect it to the app via Auth0 Token Vault.

## Using a card set

1. Pick a set folder (e.g. `shonen-anime/`)
2. Upload the images to a folder in your Google Drive
3. In the game's Settings page, enter the Google Drive folder name
4. Play!

## Adding a new set

Create a new folder here with **at least 7 images** (required for hard mode). Include a `PROMPT.md` if the images were AI-generated so others can reproduce or remix the set.
